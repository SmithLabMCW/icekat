An Interactive Fitting Tool for Continuous Enzyme Kinetics

Interact with the widgets on the left to select individual samples to optimize linear fits, select blank sample to subtract and update the model fit. Enter Python format equations* to apply a transform to the model fit. The slider and text input boxes below the progress curve residual plot can be used to refit based on a new X-axis range.

Use the figure tools at the upper right of each plot to pan
, wheel zoom
, refresh the plot
, download plot to image file
and hover over the circles on the model fit plot to see sample information
.


Click the green button to upload data files of your own. Click the blue buttons to download the slope table to a csv file or save it to the clipboard.

(Note: input data must be in csv format and is expected to be in columns with headings containing numbers denoting substrate/inhibitor concentrations (or "blank" for sample to subtract). Some special characters may not be supported and could cause an error during file upload. The "Time" column should contain integer or float values (not date-time).

*See https://docs.python.org/3/tutorial/introduction.html for information about basic python numerical operators
**The Python math library is also supported (https://docs.python.org/3.0/library/math.html)
